# 웹채팅

## 사용언어 및 실행환경
* EJS
* JavaScript (node.js 14.15.1)  
  * ejs: 3.1.5
  * fs: 0.0.1-security
  * express: 4.17.1
  * https: 1.0.0
  * peerjs: 1.3.1
  * socket.io: 3.0.3
  * uuid: 8.3.1
* Docker 20.10.17

## 기능
* uuid를 사용한 url로 서로 다른 방 입장(방 id 입력 시 같은 방 입장)
* 화상 및 음성 통화
* 채팅
* 이름 변경
